# My first blog
このサービスは私が初めて作ったブログサイトです。

実装している内容は以下の通りです：

- 記事一覧表示機能
- 記事詳細表示機能
- 記事投稿機能
- コメント機能
- 管理ユーザ登録機能
- 管理ユーザログイン・ログアウト機能
- 画像ファイルアップロード機能

## 構築環境
- WebFrameWork -> Django
- DB -> SQLite
- Server, Domain -> pythonanywhere.com
## サイトのURL
http://kazuhira1073.pythonanywhere.com/blog
